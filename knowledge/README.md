# knowledge

## 定位
全局知识库：沉淀跨前后端共享的规则、架构决策、产品方向与执行手册。

## 目录说明
- `ai/`: AI 协作规则与技能清单。
- `product/`: roadmap 与 PRD。
- `architecture/`: 系统总览与 ADR。
- `conventions/`: 代码、UI、API 约定。
- `spec/`: 全局验收标准。
- `playbooks/`: feature/API/内容/发布/维护工作手册。
- `anti-patterns/`: 全局反模式与复盘留痕。

## 维护原则
- 每个主题单一主文件，避免文档膨胀。
- 每轮迭代至少更新一条规则或反模式。
